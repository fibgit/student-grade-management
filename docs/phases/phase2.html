Phase 2: Discovery

üìÖ November 1 - January 10, 2026

Overview

The Discovery Phase focused on comprehensive requirements gathering, stakeholder engagement, and risk assessment for the St Glendale College cloud migration project. This phase laid the foundation for all subsequent architecture and implementation decisions.

Key Achievements

  • 9 comprehensive deliverables completed
  • 92 total requirements identified (40 functional, 52 non-functional)
  • 32 risks assessed and documented
  • RACI matrix established for clear accountability
  • Architecture inputs catalogue created

Methodology

Following the Solution Architecture Foundations framework, this phase employed structured interviews, documentation analysis, and collaborative workshops to gather comprehensive requirements across all stakeholder groups.

Stakeholder Engagement

Key stakeholders interviewed included:

  • CFO and Finance Team
  • IT Director and Infrastructure Team
  • Registrar and Student Services
  • Academic Department Heads

Requirements Categories

Requirements were organized into five major categories:

Accounting System

Core financial operations including AP/AR, month-end close, and financial reporting

Student Information

Student data management, enrollment tracking, and academic records

Infrastructure

Server requirements, network architecture, and storage needs

Security & Compliance

Data protection, access control, audit requirements, and regulatory compliance

Integration

System interconnections, data flow, and API requirements

Well-Architected Framework Alignment

All requirements were mapped to the AWS Well-Architected Framework pillars (CROPSS):

  • Cost Optimization: Budget constraints and cost monitoring requirements
  • Reliability: High availability and disaster recovery needs
  • Operational Excellence: Monitoring, automation, and operational procedures
  • Performance Efficiency: Response time and throughput requirements
  • Security: Data protection, encryption, and access control
  • Sustainability: Resource optimization and environmental considerations
