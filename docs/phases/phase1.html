Solution Vision

The solution vision established a clear direction for the project:

"Transform St Glendale College's IT infrastructure by migrating to AWS cloud services, following the Well-Architected Framework to ensure a secure, reliable, performant, and cost-effective solution that supports the institution's educational mission."

Stakeholder Identification

Key stakeholders were identified across multiple organizational levels:

Executive Level

  • President/Principal - Strategic oversight
  • CFO - Budget approval and financial planning
  • IT Director - Technical leadership

Operational Level

  • Finance Team - Accounting system users
  • Registrar - Student information system
  • IT Infrastructure Team - Technical implementation
  • Academic Departments - End users

Project Scope

The scope was carefully defined to ensure clear boundaries:

In Scope

  • Migration of accounting system to AWS
  • Student information system cloud deployment
  • Infrastructure as Code implementation
  • Security and compliance framework
  • Monitoring and operational excellence setup

Out of Scope

  • Application re-development (unless part of re-architect strategy)
  • Hardware procurement (cloud-based solution)
  • End-user device management

Success Criteria

Clear success criteria were established to measure project outcomes:

  • Successful migration with zero data loss
  • 99.9% uptime achievement
  • 30% reduction in operational costs
  • Security compliance certification
  • User satisfaction improvement
