About This Journey

From learning to implementation: A comprehensive AWS Solutions Architect path

The Project

This portfolio documents a comprehensive AWS Solutions Architect certification journey centered around a real-world cloud migration project for St Glendale College. The project combines theoretical learning with hands-on implementation, following professional methodologies and best practices.

My Role: Boma Fiberesima, Solution Architect

As the Solution Architect for this project, I'm responsible for:

  • End-to-end architecture design and documentation
  • Migration strategy development using the 6 R's
  • Infrastructure as Code implementation (Terraform + Ansible)
  • Application of AWS Well-Architected Framework (CROPSS)
  • Stakeholder engagement and requirements gathering
  • Risk assessment and mitigation planning

The Learning Approach

Rather than just studying for certification, this project takes a practical approach:

  1. Build a Real Application: Student Grade Management System in Java/Spring Boot
  2. Follow Professional Methodology: Solution Architecture Foundations framework
  3. Apply AWS Best Practices: Well-Architected Framework throughout
  4. Implement Modern DevOps: Terraform, Ansible, Docker, CI/CD
  5. Document Everything: Complete Project Documentation & Deliverables

Why This Approach Works

This project-based learning approach offers several advantages:

  • Real-world Experience: Not just theory - actual implementation
  • Portfolio Development: Tangible proof of skills for employers
  • Comprehensive Skill Set: Architecture + Development + DevOps
  • Professional Practices: Following industry-standard methodologies
  • Certification Ready: Preparation aligned with AWS exam domains

Location

Based in Greater London, England, UK - serving the London metropolitan area and beyond.
