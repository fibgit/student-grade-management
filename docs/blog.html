Today marks the beginning of an exciting journey into cloud architecture. I'm embarking on the AWS Solutions Architect certification path through a hands-on project: migrating St Glendale College's infrastructure to AWS.

Project Overview

The St Glendale College Cloud Migration project will serve as my practical learning laboratory. Rather than just studying theory, I'll be applying AWS concepts to real-world scenarios including:

  • Discovery and requirements gathering
  • Architecture design following the Well-Architected Framework
  • Three migration strategies (Rehost, Re-platform, Re-architect)
  • Infrastructure as Code with Terraform and Ansible
  • Cost optimization and performance tuning

What's Next

The immediate focus is on completing the Discovery Phase, which involves stakeholder interviews, requirements gathering, and initial architecture planning. I'll be documenting everything along the way to create a comprehensive portfolio.
